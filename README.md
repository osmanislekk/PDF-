# PDF Reader Application

## Overview
The PDF Reader Application is a cross-platform mobile app designed for seamless reading of PDF files on both iOS and Android devices. The app offers a simple, intuitive user interface and efficient performance for handling various PDF formats.

## Features
- **Cross-Platform Compatibility**: Available on both iOS and Android.
- **User-Friendly Interface**: Simple navigation and easy access to features.
- **Annotations**: Users can highlight text, add notes, and draw on PDFs.
- **Bookmarking**: Easily bookmark pages for quick access later.
- **Search Functionality**: Quickly find specific text within PDF documents.
- **Offline Access**: Read downloaded PDF files without an internet connection.
- **Document Organization**: Users can organize their PDF files into folders.

## Technology Stack
- **Frontend**: Flutter for building cross-platform interfaces.
- **Backend**: Firebase for authentication and cloud storage.
- **PDF Rendering**: PDF.js library for rendering PDF documents.

## Installation Instructions
### Prerequisites
- **Flutter SDK**: Ensure the Flutter SDK is installed on your machine.
- **Android Studio** (for Android development)
- **Xcode** (for iOS development)

### Steps to Install
1. Clone the repository:
   ```bash
   git clone https://github.com/osmanislekk/PDF-
   cd PDF-
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```

## Usage
After installation, open the app and load a PDF document to start reading. You can navigate pages, add annotations, and bookmark your favorite sections.
